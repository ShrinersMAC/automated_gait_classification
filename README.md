Gait Classification Code

This code will perform automated gait classifcation based on systems published by 
1. Sangeux M, Rodda J, Graham HK. Sagittal gait patterns in cerebral palsy: the plantarflexor-knee extension couple index. Gait Posture. Feb 2015;41(2):586–91. Doi:10.1016/j.gaitpost.2014.12.019
![RG](https://github.com/user-attachments/assets/1d998341-b506-4ce8-8096-12e01a5206aa)

2. Rozumalski A, Schwartz MH. Crouch gait patterns defined using k-means cluster analysis are related to underlying clinical pathology. Gait Posture. Aug 2009;30(2):155–60. doi:10.1016/j.gaitpost.2009.05.010
![RS](https://github.com/user-attachments/assets/c58a7ea4-0d32-4ff4-ad19-d25dd74c4377)


Requirements:
MATLAB 2024a or newer

Getting Started:

1. Download all files, including example .gcd files to the same folder. 
2. Run SC_gait_classification.m
3. Select the .gcd file that contains the gait data to analyze. Example files are included.
4. Each classification will be output to the command window and to a figure.

Note: Sample normative kinematic curves from our institition are provided. These normative curves can be replaced with your lab's normative values.
